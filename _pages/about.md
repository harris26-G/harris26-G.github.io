# Hanze Guo

I work on large language models, value alignment, causal reasoning, and explainable recommendation.

---

## Selected Publications (CCF A Venues)

### 1. Counterfactual Reasoning for Steerable Pluralistic Value Alignment of Large Language Models  
**NeurIPS 2025 (CCF A, Poster)**  
Hanze Guo, Jing Yao, Xiao Zhou, Xiaoyuan Yi, Xing Xie  
[arXiv](https://arxiv.org/abs/2510.18526)  

We propose **COUPLE**, a counterfactual reasoning framework for steerable pluralistic value alignment of large language models, supporting fine-grained control over pluralistic human values via structural causal models and counterfactual sampling.
