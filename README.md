# Hanze Guo Academic Homepage

This repository hosts Hanze Guo's academic homepage at [https://harris26-G.github.io](https://harris26-G.github.io).

The site is based on Academic Pages and contains:

- Research profile and selected publications
- CCF-A publication list
- Internship and education experience
- CV and structured CV data

## Local Preview

Install the Ruby dependencies and run:

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open [http://localhost:4000](http://localhost:4000).
